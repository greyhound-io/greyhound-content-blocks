The Content Block System for all Greyhound-enabled sites.

== Installation ==

1. Unzip `greyhound-content-blocks.zip` to the `/wp-content/plugins/` directory, or upload within Wordpress.
2. Activate the plugin through the 'Plugins' menu in WordPress
